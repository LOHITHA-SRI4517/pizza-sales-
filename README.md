# 🍕 Pizza Sales Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive Power BI dashboard built to analyze pizza sales data and extract meaningful business insights. The dashboard helps in understanding sales performance, customer preferences, and revenue trends.

---

## 🚀 Key Features
- 📈 Total Revenue Analysis
- 🛒 Total Orders Overview
- 🍕 Top & Bottom Performing Pizzas
- 📅 Sales Trends (Monthly & Quarterly)
- 💰 Average Order Value Insights
- 📊 Category-wise and Size-wise Analysis

---

## 🛠️ Tools & Technologies
- Power BI
- MS Excel (Data Cleaning & Preparation)

---

## 📊 Key Insights
- Identified highest revenue generating pizza: *Thai Chicken Pizza*
- Lowest performing pizza: *Brie Carre Pizza*
- Peak sales observed during specific quarters
- Average order value calculated for business optimization

---


## 📂 Dataset
- Contains pizza sales records including:
  - Order details
  - Pizza categories
  - Revenue data
  - Quantity & pricing
